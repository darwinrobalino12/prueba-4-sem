# Galería Interactiva - Semana 5

Este proyecto es una aplicación web dinámica creada con **JavaScript vanila**. Permite gestionar una galería de imágenes de forma interactiva manipulando el DOM.

##  Funcionalidades
* **Agregar Imágenes:** Mediante una URL externa.
* **Selección Inteligente:** Al hacer clic en una imagen, se resalta y se deselecciona la anterior.
* **Eliminación:** Borra del sistema la imagen que esté seleccionada actualmente.
* **Atajos de Teclado:** Soporta el uso de la tecla `Delete` o `Backspace` para eliminar.

##  Tecnologías utilizadas
* HTML5 (Estructura semántica)
* CSS3 (Flexbox y efectos visuales)
* JavaScript (Manipulación del DOM y Event Listeners)
